# Neon Survivor 🎮

ブラウザだけで遊べる、ネオン風サバイバー系アクション。
動くだけで攻撃は自動。敵を倒してレベルアップ → 3つの強化から1つ選んでどんどん強くなろう。

**ファイルは3つだけ**（`index.html`＝骨組み・`style.css`＝見た目・`game.js`＝ゲーム本体。画像も音声ファイルも使わず、効果音もコードで生成）。
ネット接続もインストールも不要。`index.html` のダブルクリックで遊べます（3ファイルは同じフォルダに置くこと）。

---

## 遊び方（操作）

| 操作 | キー / 入力 |
|---|---|
| 移動 | `W` `A` `S` `D` または 矢印キー |
| 移動（マウス派） | 画面を**長押し**するとその方向へ移動 |
| 攻撃 | **自動**（いちばん近い敵をねらって撃つ） |
| レベルアップ時の選択 | カードを**クリック** または `1` `2` `3` キー |
| 効果音 オン/オフ | `M` キー |
| もう一度遊ぶ | ゲームオーバー画面で**クリック** または `R` キー |

### 強くなる流れ
1. 敵を倒すと緑の**XPジェム**が落ちる → 近づくと自動で吸い込む。
2. ジェムが溜まると**レベルアップ**。3つの強化から1つ選べる。
3. 新しい武器（**オービット**＝体の周りを回る光弾／**オーラ**＝周囲を焼く範囲攻撃）を取ったり、ボルト（初期武器）やステータスを強化したり。
4. 時間が経つほど敵は増えて強くなる。約75秒ごとに**ボス**（トゲ付きの大きな敵）が出現。
5. HPが0になるとゲームオーバー。**生存時間 × 撃破数**でスコアが決まり、最高記録はブラウザに保存される。

## 起動方法（WSL / Ubuntu）

WSLのターミナルで、このフォルダに移動してから次を実行すると、Windows側の既定ブラウザで開きます。

```bash
cd /root/dev/projects/neon-survivor
explorer.exe index.html
```

> `explorer.exe` は Windows のプログラムで、WSL内のファイルを渡すと Windows のブラウザで開いてくれます。

うまく開かない場合は、フォルダを開いて `index.html` をダブルクリックでもOKです。

```bash
explorer.exe .
```

## 開発用メモ（おまけ）

URLの末尾に付けると動きを変えられる検証用スイッチです（ふつうに遊ぶときは不要）。

| パラメータ | 効果 |
|---|---|
| `?autostart=1` | タイトルを飛ばして即開始 |
| `?demo=1` | プレイヤーを自動操作（眺める用） |
| `?warp=秒数` | 起動時にその秒数ぶん一気に進めた状態から始める |
| `?stats=1` | 敵の数や処理時間を画面左下に表示 |

例: `index.html?autostart=1&demo=1&stats=1&warp=60`

`_shots/` フォルダは開発中に撮った確認用スクリーンショットです（遊ぶのに必要なものではありません）。
